Introdução

Esse projeto deve tomar como exemplo outros streamings para estudos já implementados, mas com o diferencial de um ambiente parecido com streamings de entretenimento, com isso fazendo o usuário ter um local de estudos mais leve e produtivo.

Performance

O Usuário terá uma vasta gama de cursos para estudar, além de um sistema usual com uma boa performance, tudo para o usuário aproveitar o máximo do conteúdo disponível para estudo sem queda de desempenho.

Qualidade

É uma aplicação streaming para web, Desenvolvido em javascript, PHP, HTML5, CSS3, Bootstrap, Integrado ao banco de dados Mysql, Jquery, Permite o cadastro de usuários, Acesso garantido por uso de senha individual.

Usabilidade

O ambiente da aplicação será operado na Web com um Layout responsivo para atuar tanto em desktop quanto em mobile, sendo assim poderá ser usado em Sistemas Operacionais com navegador compatível com a aplicação.

Ambiente do Aluno

Front end

O front-end é responsável por coligir a entrada do usuário em várias formas e processá-la para adequá-la a uma especificação em que o back-end a possa utilizar.

Back end

O back-end, parte de suporte ou parte de retaguarda são termos generalizados que se referem às etapas inicial e final de um processo.
